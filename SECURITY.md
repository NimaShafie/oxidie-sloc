# Security Policy

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Instead, report vulnerabilities privately to the repository owner through GitHub security reporting if enabled, or by contacting the maintainer directly.

Maintainer: **NimaShafie**
Repository: **https://github.com/NimaShafie/oxide-sloc**

Include:
- affected version or commit
- reproduction steps
- impact assessment
- any suggested mitigation

## Supported Versions

Until the first stable release, security support is best-effort for the latest `main` branch only.
